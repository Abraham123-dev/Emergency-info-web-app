# Emergency Contacts App

A simple web application for managing emergency contacts and accessing first aid guidelines.

## Features

- **Emergency Contacts Management**: Add, view, and organize emergency contacts with categories (Family, Medical, Friends, Work, Emergency Services, Other).
- **First Aid Guidelines**: Access step-by-step guides for common emergencies like CPR, Fire, Severe Weather, and Severe Bleeding.
- **Responsive Design**: Works on desktop and mobile devices.
- **Local Storage**: Contacts are saved locally in the browser.

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

## Setup Instructions

1. Clone or download the project files.
2. Open `index.html` in a web browser to start the application.
3. No additional setup or dependencies required.

## Usage

### Adding a Contact
1. From the home page, click "Add Emergency Contact".
2. Fill in the contact details: Name, Phone Number, Category, and optional Note.
3. Click "Save Contact" to add the contact. A toast notification will confirm the save.
4. You will be redirected to the emergency contacts list.

### Viewing Contacts
1. From the home page, click "View Emergency Contacts".
2. See all saved contacts with their details.
3. Use the call button to dial the contact (on mobile devices).

### Accessing Guidelines
1. From the home page, click "Emergency Guidelines".
2. Browse through different emergency categories.
3. Click on a guide to expand and view step-by-step instructions.
